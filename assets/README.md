# Portfolio Assets Folder

Place your files here:

## avatar.jpg
Your profile photo. Recommended size: 400×400px (square/circle will be cropped).

## projects/ folder
Place your project screenshots here. Reference them in data.json like:
- `"image": "assets/projects/ecommerce.jpg"`

## Recommended image sizes:
- Avatar: 400×400px
- Project screenshots: 1280×720px (16:9 ratio)
